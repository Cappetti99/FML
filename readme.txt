FML - Fondamenti di Machine Learning
Descrizione
Questo progetto contiene materiale didattico relativo al corso di Fondamenti di Machine Learning, con notebook Jupyter per esercizi pratici e teoria. Il progetto include due principali sezioni: i preliminari e il primo laboratorio.

Contenuto
lab1/Lab1.ipynb: Notebook del primo laboratorio.
preliminaries/Preliminaries.ipynb: Notebook dei concetti preliminari.
